\section{Accounting for Multiple Levels of Education}
\label{sec:direct-effects}

Set up the empirical model, where the education mediator has 3 levels:
$D_i^1, D_i^2, D_i^3$ which represent GCSEs, A-Levels, and a university degree respectively.
Note that these are sequential education qualifications, where GCSEs must be attained to advance to A-Levels and similarly for A-Levels to university --- $D_i^{k'} =1$ implies $D_i^{k'-1} =1$.

This leads to a mediation model, where resulting occupation-based wages and income are a result of sequential choices,
\begin{align}
    D_i^{k'} &= \phi_{k'} +
        \bar\pi_{k'} Z_i + \eta_{k',i}, \;\;\;\; \text{ for } k' = 1, 2, 3 \\
    Y_i &= \alpha + \gamma Z_i +
        \sum_{k'=1,2,3} \left(
            \beta_{k'} D_i^{k'} + \delta_{k'} Z_i D_i^{k'} \right)
        + \vec\varphi' \vec X_i
        + \varepsilon_i.
\end{align}

Describe how it is not identified, because education qualifications are not randomly assigned.
